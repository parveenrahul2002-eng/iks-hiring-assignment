## Project Overview
This is a full-stack Academic Progress Tracker application built using Spring Boot (Backend) and React (Frontend).

---

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React.js
- Database: PostgreSQL
- Authentication: JWT
- Build Tool: Maven

---

## How to Run the Project

### Backend
1. Open the backend in IntelliJ
2. Configure PostgreSQL
3. Update application.properties
4. Run TrackerApplication.java

### Frontend
1. Open frontend folder
2. Run:
npm install
npm start

---

## Key Features
- User Registration
- User Login
- Academic Progress Management
- Secure Authentication
